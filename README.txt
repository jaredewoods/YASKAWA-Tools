# Alarm Code Lookup App

## Description
The Alarm Code Lookup App is a graphical user interface (GUI) application built in Python using Tkinter and ttk libraries. It provides a convenient way to look up Alarm codes, their descriptions, and associated information.

## Features
- **Alarm Code Selection**: Users can select an Alarm code from a dropdown list of available Alarm codes.
- **Sub-Code Selection**: Once an Alarm code is selected, users can further refine their search by selecting a sub-code from a dropdown list of available sub-codes.
- **Display Information**: The application displays detailed information related to the selected Alarm code and sub-code, including message, location, signal, cause, potential causes, and sub-code description.
- **User-Friendly Interface**: The interface is designed to be intuitive and easy to use, with clear labels and organized layout.

## How to Use
1. **Launch the Application**: Run the Python script `alarm_analyzer.py` to launch the Alarm Code Lookup App.
2. **Select Alarm Code**: Choose an Alarm code from the dropdown list labeled "ALARM".
3. **Select Sub-Code**: If available, select a sub-code from the dropdown list labeled "SUB-CODE".
4. **View Information**: Detailed information about the selected Alarm code and sub-code will be displayed in the respective fields.
5. **Explore Potential Causes**: If provided, potential causes related to the Alarm are listed in the text area labeled "Potential Causes".
6. **Learn Sub-Code Description**: If available, a description of the sub-code is provided in the text area.

## Dependencies
- Python 3.x
- Tkinter (Python's standard GUI library)
- ttk (Tkinter themed widget set)

## Installation
1. Clone or download the repository to your local machine.
2. Make sure you have Python installed on your system.
3. Navigate to the directory containing the `alarm_analyzer.py` file.
4. Run the script using the command `python alarm_analyzer.py`.
